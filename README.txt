Simple ArcGIS JS API app for clicking on map and opening Google Street View.

How to use:

1. Host index.html on a web server or free hosting service like GitHub Pages or Netlify.
2. Embed the URL of the hosted page into ArcGIS Experience Builder using the Embed widget.
3. Users click anywhere on the map to open Google Street View in a new tab.

You can customize:
- Initial map center and zoom in index.html (lines 21-24).
- Basemap style by changing the 'basemap' property.
